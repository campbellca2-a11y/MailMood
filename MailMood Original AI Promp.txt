MailMood - Product Context Brief
Overview
MailMood is a browser extension (initially Chrome/Firefox) that provides emotional intelligence for email communication. It analyzes both incoming and outgoing emails to show emotional tone/mood, helping users reduce email anxiety and prevent communication mistakes.
Core Problem Being Solved

Email anxiety - People dread opening emails because they don't know if the content will be angry, urgent, or stressful
Tone blindness - People send emails without realizing they sound hostile, pushy, defensive, or overly apologetic
Relationship damage - Poorly-worded emails damage professional and personal relationships
Information overload - Can't effectively triage 50+ unread emails by subject line alone

Technical Architecture
Core Technology: MightyTagsâ„¢
Proprietary emotional parsing engine that:

Distinguishes emotional language from symptom language
Identifies specific emotional tags (frustration, gratitude, urgency, defensiveness, etc.)
Assigns intensity levels (low/medium/high)
Categorizes overall tone (Professional, Hostile, Warm, Defensive, etc.)

How It Works:
For Incoming Emails:

Extension scans email preview text in inbox view
Sends preview text to MightyTags API (over internet)
API analyzes and returns emotional tags
Extension displays color-coded indicator next to subject line

For Compose Mode:

User types email in Gmail compose window
Extension sends draft text to MightyTags API as user types (real-time)
API analyzes emotional tone in memory (doesn't store content)
Returns emotional breakdown
Extension displays tone panel with suggestions
Updates in real-time as user edits

Privacy Architecture:

Email content analyzed in real-time (in RAM/memory)
No permanent storage of email content
Process and forget model (like ephemeral analysis)
User trust built on "we never store your emails"
Optional local-only analysis mode for sensitive emails
